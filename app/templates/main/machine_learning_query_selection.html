Auf dieser Seite können Sie auswählen, auf welche Zielgruppe (Ziel) die Prognosen angewendet werden sollen. Ein "Ziel" entspricht einer bestimmten Prognose, wie beispielsweise Studienabbruch oder die Zulassung zum Master (MasterZulassung). Der "Risk Score" (Risikobewertung) gibt an, wie wahrscheinlich es ist, dass ein Student dieses Ziel erreicht oder nicht erreicht. Nachdem Sie "Ziel auswählen" anklicken, werden die Studierenden auf der nächsten Seite basierend auf ihrem "Risk Score" sortiert. Die Studierenden mit dem höchsten "Risk Score" werden zuerst angezeigt.
